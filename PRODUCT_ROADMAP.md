# MOSS - Quick Notes App: Product Roadmap

## Vision
A minimal, beautiful, lightning-fast note-taking app for macOS that lets users capture thoughts instantly without friction.

---

## Phase 1: Core MVP (Current)
**Goal:** Nail the core capture + quick access experience

### Features
- ✅ Global hotkey activation (Option+Space)
- ✅ Instant capture (Type → Save)
- ✅ Auto-detection (Notes, Reminders, Meetings)
- ✅ Quick library & search
- ✅ Tags system
- ✅ Dark/Light mode
- ✅ Edit existing notes
- ✅ Filter by type (Notes/Reminders/Meetings)
- 🔄 **Rich text formatting** (Minimal pill UI)
  - Bold, Italic, Underline, Strikethrough
  - Highlight/color picker
  - Lists (bullet, numbered)
  - Links
  - Code blocks
- 🔄 **Shift+Enter for multi-line**
- 🔄 **Text formatting quick menu** (Floating pill)

---

## Phase 2: Power User Features
**Goal:** Make it competitive with Notion/Bear

### Advanced Formatting
- Heading levels (H1-H6)
- Text alignment
- Quote blocks
- Custom fonts
- Background colors
- Inline code

### Smart Capture
- Voice-to-text notes
- Screenshot + OCR
- Drawing canvas
- Template system
- Recurring reminders with natural language
- Calendar preview on hover

### Organization
- Nested folders
- Smart collections (last 7 days, pinned, etc.)
- Advanced filtering (date range, tag combinations)
- Sort options (created, modified, alphabetical)
- Favorites/stars

### Search Enhancement
- Full-text search
- Search history
- Saved searches
- Filter by type, date, tags
- Quick filters from toolbar

---

## Phase 3: iCloud Sync & Sharing
**Goal:** Multi-device sync & basic sharing

### Sync & Storage
- iCloud sync (notes across Mac/iPad/iPhone)
- Local-first architecture (works offline)
- Automatic backups to iCloud
- Version history (restore previous versions)

### Sharing & Export
- Share notes via link (read-only)
- Export to PDF, Markdown, Plain text
- Copy formatted text to clipboard
- Apple Reminders app sync (two-way)
- Apple Calendar integration (read reminders)

---

## Phase 5: Premium Monetization
**Goal:** Sustainable revenue

### Moss Free
- Unlimited notes
- Rich text formatting
- Core features
- Local storage only
- Basic search & filters
- Dark/Light mode

### Moss Pro ($2.99/month OR $9.99 Lifetime)
- iCloud sync
- Voice input/transcription
- Custom themes & fonts
- Collaboration & sharing
- Advanced search & filters
- Image attachments
- Priority support

---

## Competitive Advantages
1. **Speed**: Instant capture (< 200ms) - fastest in class
2. **Privacy**: Local-first, iCloud only (no third-party servers)
3. **Simplicity**: Minimal design, no bloat, no AI gimmicks
4. **Affordability**: $9.99 lifetime purchase (best price in market)
5. **Smart defaults**: Auto-detection of note types
6. **Mac-native**: Optimized for macOS, works offline
